# CarbonSense Dashboard

A frontend dashboard for monitoring factory carbon emissions and energy consumption. Built with React, Vite, Tailwind CSS, and Recharts.

## Features

- **Role-based Authentication**: Mock login for Factory Owners and Sustainability Officers.
- **Factory Owner Dashboard**: View energy consumption, carbon emissions, and register a factory.
- **Sustainability Officer Dashboard**: Compare emissions across multiple factories with interactive charts.
- **Responsive Design**: Built with Tailwind CSS for a modern, clean UI.

## Tech Stack

- React (with Vite)
- JavaScript
- Tailwind CSS
- Recharts
- React Router DOM
- Axios
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Roles for Login

- **Factory Owner**: Can create a factory and view its specific metrics.
- **Sustainability Officer**: Can view and compare all factories.

*Note: This is a frontend-only prototype. All data is currently mocked and API calls are placeholders.*
